# index-cns-notes

本仓库用于归档和发布多个独立 HTML 页面，内容以个人笔记、资料整理为主，不针对任何特定网站或域名。

## 项目简介

这是一个静态 HTML 页面集合，主要用于存放和展示各类独立页面。页面内容涵盖技术笔记、资料汇总等，采用纯前端方式呈现，无需后端支持。

## 目录说明

- `pages/` — 存放所有独立 HTML 页面文件
- `assets/` — 页面用到的公共资源（CSS、JS、图片等）
- `index.html` — 仓库首页，可查看页面列表

## 页面归档说明

- 每个页面均为独立 HTML 文件，可直接在浏览器中打开
- 页面之间无依赖关系，可单独使用或引用
- 内容定期更新与整理，历史版本保留在 Git 提交记录中

## 维护说明

- 欢迎通过 Issues 提交建议或反馈
- 如需新增或修改页面，请提交 Pull Request
- 维护者会不定期检查并合并合理的贡献

## 使用方式

1. 克隆仓库到本地
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件
3. 也可通过 GitHub Pages 等服务进行在线访问
